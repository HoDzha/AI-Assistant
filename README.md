# Freelance Flow AI

Учебный проект: ИИ-менеджер задач для фрилансеров с отдельными `backend` и `frontend`.

## Структура

- `backend/app` — FastAPI backend, бизнес-логика, OpenAI-клиент и типизированные модели.
- `frontend` — статический web-интерфейс для ввода задач и отображения результатов.
- `example` — исходный demo-пример, оставлен без изменений как референс.

## Возможности

- ввод задач с проектом, статусом и GitHub-ссылкой;
- анализ списка задач через OpenAI API;
- повторные попытки запросов к API до 3 раз с экспоненциальной задержкой;
- приоритизация задач;
- формирование плана выполнения;
- сводка по проектам и их статусам;
- конфигурация через `.env` или переменные окружения.

## Запуск

1. Создай `.env` на основе `.env.example` и укажи `OPENAI_API_KEY`.
2. Установи зависимости:

```bash
python -m pip install -e .[dev]
```

3. Запусти backend:

```bash
uvicorn app.main:app --app-dir backend --reload
```

4. Запусти frontend любым статическим сервером, например:

```bash
python -m http.server 5500 --directory frontend
```

5. Открой `http://127.0.0.1:5500`.
